Html e Css


  • O que são?

Html e Css são linguagens usadas na construção de páginas na internet, que atuam de forma complementar.

A web já é parte integrante do dia a dia das pessoas e a divisão entre mundo real e mundo virtual é cada dia menor. Hoje, a facilidade de acesso permite que qualquer pessoa gere conteúdo. Por isso, tão fundamental quanto ter um site ou um portal na internet, é ter essas páginas construídas de forma organizada e atrativa, para garantir que a imagem da marca, seu conceito e sua filosofia sejam respeitados e transmitidos na web. E linguagens como HTML e CSS podem favorecer esses objetivos.

  • Suas diferenças:

> HTML (HyperText Markup Language, ou Linguagem de Marcação de HiperTexto) é uma linguagem de publicação, utilizada para estruturar conteúdos, sejam textos, imagens, vídeos ou áudios.

> CSS (Cascading Style Sheets, ou Folhas de Estilo em Cascata) é uma linguagem mais voltada para a estética, utilizada para formatar os conteúdos já estruturados. É responsável pela aparência da página, como o posicionamento dos elementos, cores e tudo que torne o site mais visualmente agradável para a navegação.
